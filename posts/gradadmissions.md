**Date:** December 2025

First and foremost, there is a ton of advice on the internet. I have shortlisted a few below, but I'd like to emphasize that this is merely my opinion, and it should be taken with plenty of salt. 

## Features

- Write in Markdown for easy editing
- Simple and clean formatting
- Easy to add new posts

## How to Add a New Post

1. Create a new `.md` file in the `posts/` directory
2. Name it something descriptive (e.g., `my-thoughts-on-cryptography.md`)
3. Write your post in Markdown
4. The blog page will automatically pick it up!

You can use all standard Markdown features:
- **Bold text**
- *Italic text*
- [Links](https://example.com)
- Lists
- Code blocks
- And more!
